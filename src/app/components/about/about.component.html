Sobre Pokédex

Este es un sitio de código abierto creado por:
Nicolás Rodríguez Salazar
Todos los datos relacionados con Pokémon fueron obtenidos del servicio de la PokéApi.

Este sitio fue diseñado con Angular v.11.0.5. La maquetación se realizó con CSS GRID, bajo la filosofía Mobile First Design para una mejor navegación en dispositivos móviles. Uso de Angular Material para paginación y autocompletado.
